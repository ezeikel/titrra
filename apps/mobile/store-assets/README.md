# Store assets — Titrra

Fleet convention: `docs` repo → `BRAND-STORE-ASSETS.md`. **The repo is the source of
truth for listing copy; the store consoles are a deploy target.**

- `aso.md` — ASO strategy, keyword research, positioning rationale (the *why*)
- `apple/metadata/` — canonical ASC metadata (`store titrra apple metadata pull/push`)
- `google/` — Play listing copy (`listing.json`) + play icon + feature graphic
- `screenshots/ios/` (+ `ios-ipad/`), `screenshots/android/` — final store screenshots,
  designed in Figma (brand file → Store Screenshots page), committed as uploaded
